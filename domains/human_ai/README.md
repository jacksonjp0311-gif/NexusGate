# Human-AI Domain

Purpose: study human-AI workflows, handoff protocols, authority boundaries, reflection loops, and operator ergonomics.

Allowed claims: observed workflow patterns, bounded recommendations, evidence-backed process improvements.

Blocked claims: autonomous authority, replacement of human authorization, unsupported model-understanding claims.

Required evidence: workflow source, command evidence, report evidence, human authority boundary.

Safe study modes: handoff packets, reflection reports, orchestration cards, operator flow maps.

Handoff surfaces: `reports/nexus_domain_intelligence_report_latest.json`, `reports/tui/nexus_tui_ai_handoff_latest.txt`.

Compiler gates: `.\scripts\nexus.ps1 domain`, `.\scripts\nexus.ps1 reflect`, `.\scripts\nexus.ps1 evolve`.

Claim boundary: human-AI workflow study does not grant autonomous authority.
