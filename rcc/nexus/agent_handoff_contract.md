# Agent Handoff Contract

Before patching:

1. Read root README.
2. Read repository context index.
3. Read Nexus index.
4. Read route map.
5. Read target folder README.
6. Patch smallest surface.
7. Run compiler.

After patching:

1. Update context if geometry changed.
2. Update failure modes if failure behavior changed.
3. Run compiler.
4. Preserve non-claim locks.
