# Governed Experience Engine v2.7.0

NEXUS GATE v2.7.0 separates three planes:

- Identity plane: source, policy, schemas, registry, and contracts that define the Source Epoch.
- Experience plane: recommendation, authorization, execution, effects, final evolve, validation, and learning receipts for one action.
- Plasticity plane: explicit, bounded, replay-safe calibration that may adjust future route pressure.

The engine does not grant autonomous authority. It records and verifies local governed experience so future recommendations can be informed only by evidence that survived the causal chain.

Core law:

```text
SOURCE IDENTITY IS NOT RUNTIME ACTIVITY.
NO VERIFIED EXPERIENCE, NO PLASTICITY.
ONLY AUTHORIZED CALIBRATION MAY CHANGE FUTURE ROUTING.
```

v2.7.0 is conservative by design. A blocked experience is useful evidence; it prevents runtime activity from becoming false learning.
