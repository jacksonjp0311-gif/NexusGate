# NEXUS GATE v2.8.0 Cybernetic Field Engine

NEXUS GATE v2.8.0 adds a sparse cybernetic field without converting NEXUS into a conventional neural network.

The field has three organs:

1. Breath Regulatory Integrity: semantic freshness, lifecycle pressure, trend, and bounded command IDs.
2. Geometric Conductance Lattice: interpretable electrical-flow route preference on selected graph edges.
3. Governed Telemetry Codec: read-only external sensing translated into NEXUS observation packets.

System law: pressure guides attention, conductance alters preference, telemetry describes context. None of them grant authority.
