# NEXUS T3MP3ST Portal

T3MP3ST is embedded under `T3MP3ST/` as a governed local security-lab surface.

NEXUS may expose:

- Electron green HUD reflection.
- Fixed local folder open.
- Fixed local War Room launcher.
- Spiral Core Portal `[16] T3MP3ST`.

Boundary:

T3MP3ST is authorized-use-only. NEXUS may reflect and open the fixed local UI, but it may not choose targets, authorize offensive action, exfiltrate secrets, store credentials, bypass scope, run arbitrary shell commands, or convert model output into security authority.

Recommended local route:

```powershell
cd "$env:USERPROFILE\OneDrive\Desktop\nexus-gate\T3MP3ST"
npm install
npm run server
```

Then open:

```text
http://127.0.0.1:3333/ui/
```
