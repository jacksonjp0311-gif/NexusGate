# NEXUS Toolbelt Cockpit

v0.9.7 Toolbelt Cockpit Output makes `./scripts/nexus.ps1 toolbelt` human-readable by default and preserves `toolbelt-json` for machine packets.

## Commands

```powershell
.\scripts
exus.ps1 toolbelt
.\scripts
exus.ps1 toolbelt-json -Tag "<intent>"
```

## Wound Compression Rule

```text
stdout = smoke only
files = evidence
tail = never truth
passed gates = certificates
active wound = one repair target
```

Boundary: local read-only cockpit; no autonomous authority; no shell authority; no network; no secret access; no git write authority.
