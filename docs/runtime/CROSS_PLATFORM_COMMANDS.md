# Cross-Platform Commands

PowerShell and Bash surfaces are peer surfaces.

| Action | PowerShell | Bash |
|---|---|---|
| Compile | `.\scripts\nexus_compile.ps1` | `bash scripts/nexus_compile.sh` |
| Once | `.\scripts\nexus_once.ps1` | `bash scripts/nexus_once.sh` |
| Loop | `.\scripts\nexus_dev_loop.ps1` | `bash scripts/nexus_dev_loop.sh` |
| Promote | `.\scripts\nexus_promote.ps1` | `bash scripts/nexus_promote.sh` |
