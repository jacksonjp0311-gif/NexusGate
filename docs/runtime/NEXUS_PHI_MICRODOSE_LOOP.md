# NEXUS Phi Microdose Loop v1.1.0

The Phi Microdose Loop grants bounded local autonomy to the Phi lane without granting durable mutation authority.

```text
compress -> compile -> local Phi advisory microdose -> patch plan -> next gate
```

## Commands

```powershell
.\scripts\nexus.ps1 phi-loop -Tag "inspect current wound"
.\scripts\nexus.ps1 phi-loop-auto -Tag "compress compile advise"
```

## Autonomy Granted

```text
run allowlisted read-only gates
call local Phi through the Ollama adapter
write evidence packets
choose the next read-only gate
emit a patch plan
```

## Still Blocked

```text
no repo file mutation
no git stage
no git commit
no git push
no secrets
no external network
no arbitrary shell
no self-authorized durable changes
```
