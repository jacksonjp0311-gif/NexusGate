# NEXUS AI Loop Fabric

NEXUS AI Loop Fabric v0.9.2 lets ChatGPT/Codex act as a meta-orchestrator over named local loops.

```text
AI intent -> named loop -> local runner -> packet -> wound/resume decision -> compiler seal -> human-authorized promotion
```

Loops include `ai-orchestrator-preflight`, `wound-indexed-resume`, `impact-map`, `bounded-validation`, `compiler-wound-focus`, `docs-doctrine-preflight`, `hud-loop-sync`, and `release-seal`.

Boundary: no autonomous authority, arbitrary shell authority, git write authority, network access, secret access, memory promotion, safety proof, security proof, or correctness proof.
