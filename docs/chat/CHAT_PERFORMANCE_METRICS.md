# Chat Performance Metrics

Version: v0.1.0

This doc defines what "chat works" means for NEXUS GATE.

## Minimum viability

- UI remains responsive.
- Messages are appended to the transcript.
- The pinned output card updates.
- Offline behavior is explicit (no silent failure).

## Useful metrics

- time-to-first-token (local)
- error classification rate
- offline graceful degradation

## Claim boundary

Metrics are local evidence only.
