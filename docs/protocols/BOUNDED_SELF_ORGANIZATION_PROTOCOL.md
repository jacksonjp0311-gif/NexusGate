# Bounded Self-Organization Protocol

NEXUS may self-organize memory and recommendation pressure. It may not self-authorize.

Levels:

- Level 0: observe events and contradictions.
- Level 1: organize memory through indexing, linking, ranking, and staleness markers.
- Level 2: apply explicitly authorized bounded route calibration.
- Level 3: propose gates, cards, demotions, promotions, and pruning candidates.
- Level 4: source or policy mutation, requiring explicit human authorization and a new Source Epoch.

Self-organization is measured by repeated verified experience, lower prediction error, lower repeated-failure rate, preserved contradictions, and no governance regression.
