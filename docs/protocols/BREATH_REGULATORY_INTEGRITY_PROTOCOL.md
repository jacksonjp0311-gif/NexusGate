# Breath Regulatory Integrity Protocol

Breath is the local vital-sign regulator. It reads evidence packets, classifies semantic freshness, checks action lifecycle pressure, appends bounded pulse history, and recommends one shell-neutral command ID.

Breath may recommend the producer of a stale surface. Breath must not recommend itself as the solution to stale upstream evidence.

Breath writes local runtime state under `state/breath/`, which is not canonical source identity.
