# NEX BUILD Protocol

BUILD reconstructs the active cognitive field from Source Epoch identity, Breath, conductance, language activation, self-model evidence, lifecycle state, and authority boundaries.

Activation uses:

```text
a(k+1) = (1-d)s + dPa(k)
```

with default `d = 0.82` and at most 20 iterations. Contradiction and uncertainty reduce final activation. BUILD does not authorize action.
