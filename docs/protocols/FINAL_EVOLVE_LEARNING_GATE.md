# Final Evolve Learning Gate

Targeted validation can prove a local check passed. It cannot by itself create durable learning.

Durable learning requires:

- execution receipt
- effect receipt
- validation receipt
- action chain verification
- epoch verification
- final evolve evidence
- clean pre/post epoch admissibility
- no high-severity confounders

If final evolve is required and has not passed, finalization must produce `NOT_LEARNABLE`.
