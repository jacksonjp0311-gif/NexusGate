# Governed Agent Continuity Protocol

NEXUS GATE v2.0 defines a portable continuity protocol for human-directed AI systems.

```text
Intent Packet
-> Origin Certificate
-> Evidence Packet
-> Authority Decision
-> Human Authorization
-> Controlled Invocation
-> Execution Receipt
-> Validation Certificate or Wound
-> Memory Promotion Decision
-> Continuation Packet
```

The protocol is designed for Codex, ChatGPT, local agents, TUI, Electron, future dashboards, and governed tool surfaces.

## Boundary

The protocol may preserve orientation across sessions.
In v2.1, causal coherence routing can prioritize orientation, repair, or validation under pressure.

It may not create autonomous authority.

```text
self-observation != self-authorization
coherence != permission
recommendation != execution
```
