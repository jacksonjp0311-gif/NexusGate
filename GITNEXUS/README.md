# GITNEXUS for NexusGate

GITNEXUS is the NexusGate-native codegraph intelligence organ.

It scans repository files, identifies symbols/imports/dependencies, estimates impact,
and returns evidence packets before humans authorize durable mutation.

Boundary:
- evidence only
- no autonomous authority
- no arbitrary shell execution
- no NexusCell policy change
- no model-output file mutation
